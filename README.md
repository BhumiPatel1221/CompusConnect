# CompusConnect
CampusConnect — A full-stack College Event Management System built with React, Express.js, and MongoDB. Features JWT auth, role-based access, event &amp; registration management, company placement visits, AI-powered recommendations, real-time chatbot via Socket.IO, and an admin dashboard.

Tech Stack
Frontend: React 18 · TypeScript · Tailwind CSS 4 · Radix UI · Shadcn/UI · Recharts · React Router 7 · Vite
Backend: Node.js · Express 5 · MongoDB · Mongoose · JWT Authentication · Socket.IO · PM2

Features
Authentication & Authorization — JWT-based auth with role-based access control (Student / Admin)
Event Management — Full CRUD for college events with eligibility criteria and filtering
Event Registration — Students can register for events with duplicate prevention and feedback
Company Placement Visits — Manage company visits and student applications for placements
AI-Powered Recommendations — Personalized event suggestions based on student interests
Real-Time Chatbot — Socket.IO-powered chatbot for instant assistance
Admin Dashboard — Analytics, user management, and event oversight
File Uploads — Multer-based image/document upload support
Centralized Error Handling — Consistent API error responses
Load Balancing — PM2 cluster mode for production deployments
